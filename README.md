# bookstore

Um projeto de loja de livros feito com Django.

Este projeto visa demonstrar a criação de um backend RESTful para uma loja de livros, utilizando o framework Django e o gerenciador de dependências Poetry.

## Como Executar

Siga as instruções para configurar o ambiente de desenvolvimento.

## Funcionalidades

- Listagem de livros
- Gerenciamento de livros
